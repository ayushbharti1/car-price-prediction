# Car Price Prediction 

This project predicts the price of a car using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Dataset
Car dataset containing features like:
- Year
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission

## How to Run

1. Clone the repository
2. Install requirements
3. Run the model

```bash
pip install -r requirements.txt
python app.py

Project Files
